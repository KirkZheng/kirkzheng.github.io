Windows也能像MacOS一样丝滑

发表于 2019-07-18 | 更新于: 2021-01-23 | 分类于 Windows10的相关配置 | |
| 字数统计: 897字 | 阅读时长 ≈ 3分钟

        最近新入手了一个thinkpad，之前的mbp给卖了，买的原因就不提了，卖thinkpad的是因为预算不足，然额经过一周的上手，发现巨硬的系统是越来好用了，从mac过渡过来是一点问题没有，而且还可以白嫖很多好收费的软件，简直不要太爽！
        在mac最常用的就是全局搜索工具Alfred,确实方便，Windows平台上其实也有一个咱们国人开发的全局搜索工具Wox，用了一会儿，发现比Alfred还好用，大伙可以去官网won.one下载，注意要下载完整的安装包。
        并且安装的时候也要全部安装，所有的东西都是相辅相成的。用过你就会知道，马上就离不开了。MacOS有个终端工具iterm2挺好用，win可以用git代替，功能或许没有那么强大，满足基本需求也还是可以的。
        Windows因为其庞大的用户，所以很多软件开发商都会为其开发软件，二位因为其exe文件很容易反编译，也就是可以破解，升值一些偏门的软甲只有win平台有，我就给大家推荐个人常用的科学上网软件自由门，是臭名昭著的反共网站大纪元开发的免费翻墙工具，别说还真的挺好用，估计是免费的里面最好的了，反正我又不用来反共，俺觉着共产党治理下的祖国还是很好的。自由门需要翻墙下载，大伙们可以先去GitHub上面下载蓝灯，这个不用翻墙，安装上蓝灯再去下自由门,蓝灯不怎么稳定，速度也慢，所以我也就没推荐了，不差钱的可以使用付费的快连VPN，速度很快，价格也很实惠。
        有时我们觉着win10自带的软件太多又不怎么会用得着，我们可以使用Get-AppxPackage * | Remove-AppxPackage命令一键删除，再装上我们自己想要安装的软件，其实win10自带的应用都挺好用的，看个人喜好，我是删了的。什么问题都没有，很清爽，这也是微软允许的操作，操作方法是快捷键Windows + S，输入powershell，然后右键选择「以管理员身份运行」，输入上述命令，等待几分钟就可以了。
        另外系统必备的软件推荐一下：

  • 射手影音经典版–小巧却功能强大的视频软件
  • QQ轻聊版–精简过的QQ，官网已经下架了
  • 搜狗输入法智慧版–比原版更有智慧
  • 比特彗星–下载工具，比迅雷良心
  • Bandizip–轻便简洁的文件解压缩工具
  • Foxmail–腾讯出的第三方邮箱客户端

        浏览器用自带的就好了，注意是新版的Edge，个人感觉比Chrome更省电速度也更快。

        重大更新，我给win10装了一个终端！！！
         这几天突然有点想念MacOS的终端，寻思想想能不能在win平台整一个，谷歌一搜，还真有办法，我按照博客园的Win10安装cygwin并添加apt-cyg这篇博客装上了终端，按照zsh+on-my-zsh配置教程指南（程序员必备）【已备份】这篇文章让终端变得更好用，喜欢折腾的可以试试！！
        如果遇到无法访问github的情况可以在C:\Windows\System32\drivers\etc\hosts文件中添加199.232.68.133 raw.githubusercontent.com解决亲测有效！
