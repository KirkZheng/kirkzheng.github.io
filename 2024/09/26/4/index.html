一段时光

发表于 2024-09-26 | 更新于: 2025-02-28 | |
| 字数统计: 794字 | 阅读时长 ≈ 2分钟

这么多年过去了，还是那么恨，张元策这个又晕又蠢的杂种当初辱骂老子是野狗，故意找事以权压人，跟疯狗一样吼叫我都忍了，受不了的是后面班级已经解散了，这狗舔着逼脸私下说对不起我，我因为不搭理它，这条死狗竟然背后当着那么多人诋毁我，当面一套背后一套毫无原则，等我反应过来这猪已经跑了，我是真的恨，怎么碰到这么个蠢逼晦气玩意。

后面落到丁亚军手里，这烂人也不知道发什么神经，对我也是成见颇大，无缘无故说要打死我，硬逼我不停的抽烟、吃东西、用打火机烫我、用应急棍往我身上砸。我也不知道做错了什么，我一直都是老老实实的，干活也卖力，边上兄弟也是这么认为的，后面听张文浩说是因为丁看我不爽，以至于连闫玉鹏都看不下去。

这两个屌人拿着每月几千块包吃包住养得膘肥体胖的，天天折腾一个月几百块累死累死的实在人，天南海北聚在一起也是一种缘分，搞的别人那么恨你图啥？很开心吗？心理变态吧，傻不拉几的。我那时候刚大学毕业，没什么社会经历，心智是很单纯的，这两个屌人对我产生了很大的影响，以至于我现在一想到那段经历就痛苦万分。

我其实一直都知道，都是看人下菜，说白了就是看我不爽，当时也不敢反抗，个人性格以及周围环境迫使我百般忍耐，想着熬过去就好了。确实，到了第二年好了很多，靠着自己的能力让周围兄弟改观了很多，虽然有时还会被徐瑞峰拿第一年的经历嘲讽，但总体还是比较舒服的一年，也相对怀念这段时光。

你问我有没有后悔，我非常后悔，早知道就不去了，跟我想象的差太多。当然，在其位谋其职，里面大部分人都很好，像鲁龙伟、马荣生、刘云、吴飞飞、刘艳鹏、薛斯成、李虎、黄芳遮、闫玉鹏、綦崎超、龙建华等人，他们让我感受到了集体生活的意义和温暖，这些人我一辈子记着，让我还能回想起短短两年生活中的闪光点。

至于我为什么突然说出这段经历，主要还是因为仇恨情绪扰乱了我的心绪，我太压抑了，加上我非常记仇，所以选择通过微博宣泄一下。我并不是没想过去报复，也做出过实际行动，但后面想想也没必要了，恶人自有天收，我就不去当这个傻子了，虽说现在混的也不好，但毕竟熬过几段艰难的时光，让我成长了很多，准备迎接剩下更多的崎岖不平。
